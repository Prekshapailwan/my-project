Colorful galaxy

The color of a galaxy can indicate the types of stars it contains. Blue galaxies contain more young stars and are younger galaxies. Red galaxies contain more older stars and are thought to be older galaxies. The color of a galaxy can also indicate its star formation rate. Blue galaxies are actively forming stars, while red galaxies are more quiescent. However, dust can also make galaxies appear redder. If a galaxy contains a lot of dust, its color may depend on its orientation. In plots of u-r color index versus absolute Mr magnitude, early-type galaxies appear as a narrow red sequence. Later-type galaxies, which are mostly star-forming, appear as a broad blue sequence.
